Le programme...

Introduction
L'objectif de HTML, CSS et JavaScript, la différence entre les framework et les langages, et comment s'y retrouver dans un projet de site Web de base avec Atom.
Pages web, la base
Le squelette de chaque page web sur Internet, ainsi que les éléments HTML fondamentaux tels que les titres, les paragraphes, les listes, etc.
Liens et images
L'établissement de liens vers d'autres pages, la référence à ces dernières avec des chemins absolus, relatifs et relatifs à la racine, l'intégration d'images et la finition du squelette de base de notre page web.
Du style avec CSS
Connecter une feuille de style, styliser les éléments HTML avec toutes sortes de propriétés CSS, sélectionner différents éléments et réutiliser des styles sur plusieurs pages web.
Le modèle de la boîte
Boîtes à blocs, boîtes en ligne, padding, bordures, marges, dimensions, et une introduction à la façon dont les sites web sont construits avec des boîtes.
Les sélecteurs css
Les sélecteurs de classe, les sélecteurs de descendants, les très utiles
, les liens de style avec les pseudo-classes, et pourquoi les sélecteurs d'identification sont une mauvaise idée.
Float
Introduction à la mise en page CSS à base de float, au contrôle du flux horizontal de la page, à l'élimination des float et à la dissimulation des débordements.
Flexbox
Les conteneurs flexibles par rapport aux articles flexibles, l'alignement vertical et horizontal, la distribution, l'emballage, la réorganisation et la création de mises en page CSS modernes.
Grid
La grille ou le graâl de la mise en page moderne. Enfin une façon de positionner son contenu en deux dimensions sur votre page web, et bien plus encore.
Positionnement avancé
Le positionnement statique, relatif, absolu et fixe, les menus déroulants, l'index z et d'autres CSS sophistiqués qui vous feront sentir vraiment intelligent.
Un site Responsive
Utilisation des requêtes média pour contrôler l'application des règles CSS, développement mobile en premier et désactivation du zoom de la fenêtre de visualisation.
Images réactives
La partie difficile de la conception réactive : écrans rétiniens, images fluides, optimisation des dispositifs avec srcset, et direction artistique avec l'élément .
Html sémantique
Le plan du document, les articles, les sections, les navigations, les en-têtes, les pieds de page, les apartés, les figures, et d'autres moyens de rendre votre balisage HTML plus informatif. Plus d'informations.
Formulaires
Formulaires frontaux : le HTML pour créer des champs de texte, des boutons radio, des menus déroulants, des cases à cocher et des boutons, ainsi que le CSS pour les styliser tous.
Typographie web
Où trouver des polices de caractères pour le web, hébergement local ou externe, gestion de plusieurs polices de caractères et principes typographiques de base comme l'alignement du texte, l'orientation et la mesure.
Et si on testait
Mise en pratique et experimentations diverses si le temps le permet...
