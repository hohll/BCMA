Projet "Allo Ciné"

---------------------------------------------------------------------------------------------------------

Il s'agit de reproduire une page fictive du site "Allo Cinée", voir l'image "screen.png", située à la racine de ce zip, pour voir le rendu final attendu.

Le fichier "index.html" contient un déut de structure à compléter et modifier.
Le dossier css contient un fichier de reset.css et un fichier style.css à compléter et modifier.
Toutes les images sont dipsonibles dans le dossier "images".
