Les consignes

  • Télécharger le fichier contenant les fichiers et le dézipper sur votre PC
  • Vous avez, les images dans le dossier "site-exam/images", les fonts dans "site-exam/assets/fonts/", jquery dans "site-exam/assets/js/" et le reset css dans le fichier "site-exam/assets/css/style.css"... A vous de voir si tout est utile...
  • Pour le titre principal "LOREM" le code couleur est # et pour le bouton bleu # et pour le bouton gris #, ou passage de la souris les boutons .. et les liens ..
  • La police est par défaut Arial... et la couleur est pour les titre #, les sous-titre # et le texte #
  • Faire au moins le design 100% écran et en mode responsive le smartphone ou la tablette pour les deux pages... faire les trois une belle note assurée !
  • Vous avez le choix des outils et pour le texte vous utilisez un générateur de Lorem Ipsum de votre choix...
  • Les icones dans le footer sont issues de la police FontAwesome...
  • Le HTML et le CSS seront notés sur 10 points chacun !
